## Updating From Current Firmware

1. Open the GyroStainer web UI.
2. Go to **Settings > Software Update (OTA)**.
3. Use direct browser upload with `gyrostainer-esp32-hmi-v0.4.1.bin`, or paste the recommended OTA URL above into **Firmware URL**.
4. Click **Install Update**.
5. Wait for reboot and reconnect.
6. Confirm the software version is `esp-hmi-0.4.1`.
